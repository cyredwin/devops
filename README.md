# Cloud DevOps Engineer Nanodegree

This repository contains my work , and projects completed as part of the **Udacity Cloud DevOps Engineer Nanodegree**.

## Program Overview

The Cloud DevOps Engineer Nanodegree focuses on building, deploying, and maintaining cloud-based applications using modern DevOps practices. The program covers cloud fundamentals, infrastructure as code, containerization, Kubernetes, CI/CD pipelines, monitoring, and logging.

## Courses Included

### 1. Welcome to the Nanodegree Program
Introduction to the program structure, expectations, and available learning resources.

### 2. AWS Cloud Fundamentals
Topics covered:
- Cloud computing basics
- AWS compute services
- Storage and content delivery
- Security fundamentals
- Networking and elasticity
- Messaging and containers
- AWS management tools

**Project:**
- Deploy Static Website on AWS

### 3. Deploy Infrastructure as Code (IaC)
Topics covered:
- CloudFormation basics
- Infrastructure diagrams
- Networking infrastructure
- Servers and security groups
- Storage and databases

**Project:**
- Deploy a High-Availability Web App using CloudFormation

### 4. Microservices at Scale using AWS and Kubernetes
Topics covered:
- Microservices architecture
- Containers for DevOps
- AWS microservices
- Kubernetes fundamentals
- Operationalizing Kubernetes

**Project:**
- Operationalizing a Co-working Space Service

### 5. Build CI/CD Pipelines, Monitoring, and Logging
Topics covered:
- Introduction to CI/CD
- Continuous integration and deployment
- Building CI pipelines
- Deployment pipelines
- Monitoring and logging

**Project:**
- Movie Picture Pipeline

### 6. Congratulations!
Program completion and final wrap-up.
