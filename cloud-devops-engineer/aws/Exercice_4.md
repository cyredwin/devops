
# Exercise 4: Create a VPC

## Steps

1. Create a VPC.
2. Create a public subnet inside the VPC.
3. Create and attach an Internet Gateway to the VPC.
4. Add a route that sends `0.0.0.0/0` traffic to the Internet Gateway.


## Expected Outcome

After completing this exercise, you will have:

* A functional VPC.
* A public subnet with internet access.
* Proper routing configured through an Internet Gateway.
* A basic network setup ready for deploying AWS resources.
